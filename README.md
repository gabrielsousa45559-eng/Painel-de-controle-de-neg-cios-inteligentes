# Painel de controle de negócios inteligentes

Sistema web desenvolvido em **JavaScript avançado**, focado em **controle de vendas, caixa e análise financeira**, inspirado em cenários reais de empresas como **vendas de tratores** e **postos de combustível**.

##  Funcionalidades

- Cadastro de vendas com:
  - Produto
  - Valor
  - Vendedor
  - Comissão automática (5%)
- Controle de caixa:
  - Entradas
  - Saídas
  - Histórico
- Dashboard financeiro com:
  - Total de vendas
  - Total de comissões
  - Entradas e saídas
  - Saldo final
- Persistência de dados com **LocalStorage**
- Sistema 100% em **arquivo único**

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (DOM, lógica, arrays, reduce, filter)
- LocalStorage

## Objetivo do Projeto

Demonstrar habilidades práticas em:
- Lógica de programação
- Raciocínio matemático
- Organização de dados
- Desenvolvimento de sistemas administrativos reais
- JavaScript aplicado ao contexto empresarial

## Como usar

1. Clone o repositório
2. Abra o arquivo `index.html`
3. Comece a registrar vendas e movimentações
4. Os dados ficam salvos no navegador
